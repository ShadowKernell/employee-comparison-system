# Employee Comparison System

Version: v1.0.0

A simple C++ project created while learning C++ fundamentals.  
This project uses struct, functions, templates, and user input to compare two employees.

## Features

- Enter information for two employees
- Display employee information
- Calculate average salary
- Compare salaries
- Calculate salary difference
- Compare employee ages

## Technologies

- C++
- Struct
- Functions
- Function Templates
- Switch Statement
